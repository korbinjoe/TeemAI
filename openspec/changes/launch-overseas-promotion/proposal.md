# Launch Overseas Promotion

## Summary

Develop and execute an overseas open-source community promotion plan for OpenTeam, establishing project awareness on core channels such as Hacker News, Reddit, and Twitter/X, and acquiring the first batch of international users and contributors.

## Motivation

OpenTeam, positioned as the "AI Super-Individual's Operating System," offers unique value in the overseas AI coding tool landscape with its differentiated multi-Agent parallel orchestration approach. Currently the project only has Chinese community exposure and needs a systematic entry into English-speaking open-source communities for user growth.

## Goals

1. Achieve 500+ GitHub Stars within the first 4 weeks of launch
2. Establish project awareness across the three core channels: HN / Reddit / Twitter
3. Build a sustainable content output cadence and community interaction mechanism
4. Attract 3-5 overseas contributors to participate in development

## Non-Goals

- No paid advertising
- No ProductHunt launch (product maturity insufficient, deferred)
- Not pursuing short-term explosive growth; prioritizing reputation building

## Approach

### Core Positioning (One-liner Pitch)

> "One person, a full AI engineering team — orchestrate multiple Claude/Codex agents in parallel through a Web IDE."

### Differentiation Messaging

| vs Competitor | OpenTeam Differentiator |
|---------|----------------|
| Cursor / Windsurf | Multi-Agent parallel, not single Agent conversation |
| Claude Code CLI | Web IDE visual orchestration, continues running after you leave |
| Devin | Open source + runs locally + not locked to a provider |
| Aider | GUI + team orchestration + Workspace isolation |

### Channel Strategy

1. **Twitter/X** — Warm-up + ongoing operations (build-in-public content)
2. **Hacker News** — Launch spike (Show HN post)
3. **Reddit** — Targeted coverage across multiple subreddits
4. **Dev.to / Medium** — Long-form SEO + story distribution
5. **GitHub Ecosystem** — Awesome Lists, Discussions, Issue engagement

### Timeline

| Phase | Period | Focus |
|------|------|------|
| Phase 0: Preparation | W1 | Asset creation (Demo GIF/Video, comparison chart, blog post) |
| Phase 1: Warm-up | W2 | Twitter build-in-public post series |
| Phase 2: Launch | W3 | HN Show HN + Reddit first post |
| Phase 3: Expansion | W4+ | Cross-channel secondary distribution + community engagement + Awesome Lists |

## Risks

- HN launch may sink without traction → Fallback: drive traffic via Twitter KOL retweets
- Reddit spam detection false positive → Build account karma in advance
- Demo video quality insufficient for conversion → Validate with GIF first before investing in video
